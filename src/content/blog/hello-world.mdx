# 你好，世界！

欢迎来到我的个人博客！这里将记录我的技术探索、学习笔记和生活感悟。

## 技术栈

这个博客使用了当下最前沿的技术方案：

- **框架**：[Astro v5](https://astro.build) — 内容驱动网站的事实标准，零 JS 默认发送到客户端
- **样式**：[Tailwind CSS v4](https://tailwindcss.com) — 全新重构的原子化 CSS 框架
- **内容**：MDX — Markdown + JSX，让文章可以嵌入交互组件

## 特色功能

除了普通的博客文章，这里还有一个**网址导航**页面，收录了我日常使用的各类优质网站，分门别类地整理，方便快速访问。
